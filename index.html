  • Orca Hand Replica

    With Feetech servo and 3D printing, we (with my classmate) built a replica of the orca hand with 10% budget of the original one.
    Also I build a human demonstration project based on computer vision to control the dexterous hand.

    3D Printing Feetech Servo Human demonstration

    2025 | Role: Co-leader, Software Migration
    Explore More

    The project is based on the open-source Orca Hand from OpenAI.
    We redesigned the finger structure to fit the Feetech servos, and 3D printed the parts.
    For the control system, we adapted the original software to work with our hardware setup.
    Additionally, I developed a computer vision system to capture human hand movements and translate them into commands for the dexterous hand.
    If you are interested, please check out the vision for dexterous action (V4D) on GitHub.
